# farmadvisor-claude-plugins

Marketplace repo for various FarmAdvisor Claude plugins.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add IFC-NC/farmadvisor-claude-plugins
```

## Available Plugins

_No plugins have been added yet. See [CONTRIBUTING.md](CONTRIBUTING.md) for how to add a plugin._
